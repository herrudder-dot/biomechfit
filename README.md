# BiomechFit - 身体特性診断

質問に答えるだけであなたの身体特性がわかる、サイクリストのためのバイオメカニクス診断アプリ。

## 機能

- 🧬 4スタンス理論に基づくタイプ診断
- ⚡ APA（ペダリング特性・姿勢傾向）分析
- 🪑 タイプ×APAに基づくサドル・クリート設定ガイド
- 🚴 おすすめ機材レコメンド
- 📖 タイプ別走り方ガイド

## 開発

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

## デプロイ

Vercelに接続してプッシュするだけで自動デプロイされます。
